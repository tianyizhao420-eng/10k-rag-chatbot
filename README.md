# 📊 10-K RAG Chatbot

An interactive **Retrieval-Augmented Generation (RAG)** chatbot for analyzing **10-K financial reports** from companies such as Amazon, Google, and Microsoft.

Users can upload multiple 10-K PDF files and ask questions about financial data, business risks, or company performance.

The system retrieves relevant document sections and generates answers strictly grounded in the source documents.

---

# 🚀 Strong LLM Features

- Upload multiple **10-K PDF reports**
- Ask questions about financial data or business risks
- **RAG architecture** using:
  - OpenAI embeddings
  - FAISS vector search
  - Multi-query retrieval
- Transparent answers with **retrieved document context**
- Simple **Streamlit chat interface**
- Real-time **token streaming simulation**
